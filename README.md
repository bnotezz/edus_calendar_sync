# 🎓 School Calendar Sync (Edus.school -> Google Calendar)

Автоматизований інструмент для синхронізації шкільного життя вашої дитини з Google Календарем. Більше не потрібно заходити в додаток школи, щоб дізнатися розклад або що сьогодні на обід.

## ✨ Основні можливості

* 📚 **Розумний розклад:** Автоматичний імпорт усіх уроків із зазначенням вчителів.
* 🍽️ **Меню в кишені:** Інформація про сніданки та обіди додається прямо в опис відповідних подій у календарі.
* 🏖️ **Трекер канікул:** Автоматичне створення подій для канікул та державних вихідних.
* 🎭 **Візуальні теги (Emoji):** Швидке розпізнавання типу події за допомогою емодзі:
* 📚 — Навчальні уроки.
* 🍽️ — Харчування (із меню).
* 🏖️ — Канікули та вихідні.


* 🤖 **Повна автоматизація:** Працює через GitHub Actions — жодних серверів чи постійно увімкнених ПК.
* ⚠️ **Сповіщення про помилки:** Якщо токен школи застаріє, бот надішле вам повідомлення в Telegram.

## 🚀 Швидкий старт

### 1. Підготовка Google Календаря

1. Створіть окремий календар (рекомендується в акаунті дитини).
2. В [Google Cloud Console](https://console.cloud.google.com/) створіть проект, увімкніть **Google Calendar API** та створіть **Service Account**.
3. Завантажте **JSON-ключ** сервісного акаунта.
4. Додайте `client_email` з цього JSON у налаштування доступу вашого Google Календаря з правами "Внесення змін".

### 2. Отримання даних Edus

1. **SCHOOL_HOST:** Повна адреса вашої школи (наприклад, `https://nasha-shkola.edus.school`).
2. **SCHOOL_TOKEN:** Знайдіть `Authorization: Bearer` у Network-запитах вашого браузера після входу в систему школи.
3. **SCHOOL_USER_UUID:** ID дитини, який відображається в URL розкладу (наприклад, `21bd7513...`).

### 3. Налаштування GitHub Secrets

У вашому репозиторії перейдіть до **Settings > Secrets and variables > Actions** та додайте наступні секрети:

| Секрет | Опис |
| --- | --- |
| `SCHOOL_HOST` | Адреса вашого порталу edus.school |
| `SCHOOL_TOKEN` | Ваш Bearer токен доступу |
| `SCHOOL_USER_UUID` | UUID користувача (дитини) |
| `GOOGLE_CALENDAR_ID` | Email календаря (наприклад, `primary` або ID з налаштувань) |
| `GOOGLE_SERVICE_ACCOUNT_JSON` | Повний вміст вашого JSON-ключа Google |
| `TELEGRAM_BOT_TOKEN` | (Опціонально) Токен від @BotFather |
| `TELEGRAM_CHAT_ID` | (Опціонально) Ваш ID від @userinfobot |

## 📅 Графік роботи (Workflows)

Система налаштована на два незалежні цикли:

1. **Щодня о 04:00:** Оновлення розкладу уроків та меню на поточний тиждень.
2. **Щонеділі о 05:00:** Оновлення даних про канікули та семестри.

## 🛠️ Технології

* **Python 3.9+** — основна логіка синхронізації та обробки даних.
* **Google Calendar API v3** — взаємодія з календарем.
* **GitHub Actions** — CI/CD для регулярного запуску скриптів.
